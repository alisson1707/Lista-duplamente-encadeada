# Lista Duplamente Encadeada em TypeScript

## Integrantes
- Maria Clara Almeida Martins
- Álisson Ramires Sena da Silva

## Descrição
Este projeto implementa um TDA de Lista Duplamente Encadeada em TypeScript,
permitindo navegação do início ao fim e do fim ao início sem utilizar arrays
ou estruturas prontas.

## Funcionalidades
- Inserir no início
- Inserir no fim
- Inserir em posição específica
- Remover primeiro
- Remover último
- Remover em posição específica
- Verificar se está vazia
- Buscar valor
- Tamanho da lista
- Exibir do início ao fim
- Exibir do fim ao início
